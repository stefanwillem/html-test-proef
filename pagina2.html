Over ons

Dit is een proefproject om te leren werken met HTML en GitHub Pages.

🎯

Ons doel

Een simpele, mooie website waar je snel HTML-wijzigingen online kunt testen zonder ingewikkelde tools.

🛠️

Hoe het werkt

De site draait op GitHub Pages. Elke keer dat je een bestand aanpast en commit, wordt de site automatisch bijgewerkt.

💡

Tip

Open een HTML-bestand in GitHub, klik op het potlood-icoon, pas de tekst aan en klik op "Commit changes".

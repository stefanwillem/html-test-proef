About us

This is a test project to learn how to work with HTML and GitHub Pages.

Our goal

A simple, polished website where you can quickly test HTML changes online without complicated tools.

🛠️

How it works

The site runs on GitHub Pages. Every time you edit a file and commit, the site updates automatically.

💡

Tip

Open an HTML file in GitHub, click the pencil icon, edit the text, and click "Commit changes".
